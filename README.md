## Etapas do Projeto
1. Importação das bases
2. Retirada da média de notas das estratégias A e B
3. Cálculo de média e variancia das estratégias
4. Impressão do resultado do teste Z
5. Montagem do Gráfico da distribuição estatistica

##  Como Executar

### Pré-requisitos
- Python 3.7+
- pip ou conda

### Instalação

```bash

git clone https://github.com/GrizzoHenrique/EstatisticaAplicada.git
cd EstatisticaAplicada

# Crie um ambiente virtual (recomendado)
python -m venv venv

# Instale as dependências
